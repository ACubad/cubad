# Windows tooling note

GitHub CLI may already be installed at `C:\Program Files\GitHub CLI\gh.exe` while an existing
PowerShell session has a stale `PATH`. Before treating `gh` as unavailable or asking to reinstall
it, check that standard location and invoke the executable directly when present.
